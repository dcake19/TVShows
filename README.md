# TVShows

App lets users search and discover TV Shows and then download them to view details (overview, seasons and epsiodes, cast and recommendations)

## Demo app using:
* SQLite
* RxJava2
* Retrofit2
* Dagger2
* Espresso
* Mockito
* TMDb API
